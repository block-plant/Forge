Database Engine Manual

              
Forge implements a custom Key-Value store on top of an LSM-Tree derivative.

              
    
                
  • REST API: You can directly manipulate data by sending POST/GET requests to /db/{collection}.
    • 
                
  • Data Isolation: Each project folder in /opt/forge/projects/{slug}/data acts as a completely isolated SQLite/LSM namespace.
